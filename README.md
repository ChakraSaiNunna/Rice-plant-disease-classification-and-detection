# Rice Disease Detection using CNN + DenseNet + Bat Algorithm

This project detects and classifies rice leaf diseases using a hybrid CNN-DenseNet model with Bat Algorithm optimization.

## Features
- Uses DenseNet121 for feature extraction
- Fine-tuned with Bat Algorithm for parameter optimization
- Data augmentation for better generalization

## How to Run
1. Clone this repo
   ```bash
   git clone https://github.com/chakrasainunna/Rice-Disease-Detection.git
   cd Rice-Disease-Detection
2.Install dependencies
     pip install -r requirements.txt
3.Run the main script
     python main.py
